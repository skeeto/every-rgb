All 18-bit Colors

This canvas will slowly fill up at random with all 262,144 18-bit colors in a unique arrangement. This will take while, so go get something to drink.

The algorithm works like this:

  1. Pick a color and set the center pixel to this color.
  2. Pick another color.
  3. Find the colored edge pixel closest in color to this color.
  4. Set one of its uncolored 8-adjacent pixels to this color.
  5. Go to step 2 unless all pixels have been colored.
